# Compounding Calculator

A lightweight, responsive investment growth calculator built with a single HTML file. It supports recurring contributions, monthly or yearly compounding, multiple currencies, summary metrics, a growth chart, and monthly or yearly result tables.

## Run Locally

Open `index.html` directly in a browser, or run a local server from this folder:

```bash
python3 -m http.server 4174
```

Then visit `http://localhost:4174`.

## Publish

This project can be published directly with GitHub Pages using the `main` branch and the repository root.
